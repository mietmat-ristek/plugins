Ristek plugin samples solution
22.11.2022

-The minimum required .NET framework version is 4.8.
-See the Epx.BIM, Epx.BIM.Plugins, Epx.Ristek.Data and Epx.Ristek.Data.Models namespaces in the documentation.
-Resource dictionaries for localization must have base name Strings.resx and they must be placed in a folder called Strings under the project root.
-Resource keys can be generated using the supplied Resource Key Generator. 
-Place the built plugin assemblies in the Documents\3DTrussme\Plugins directory.
-Debugging is possible by attaching to the 3DTrussme process once it has loaded (and using plugin assemblies built in Debug mode).
